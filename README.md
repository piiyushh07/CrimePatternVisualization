# Crime Pattern Visualization

## 📌 Description
This project is a Java-based Crime Dashboard that visualizes crime data of different cities using a graphical map interface.

## 🚀 Features
- City-wise crime visualization
- Interactive UI (click on cities)
- Image-based dashboard
- Simple and user-friendly design

## 🛠️ Technologies Used
- Java (Swing/AWT)
- OOP Concepts
- Image Handling

## 📂 Project Structure
src/
├── CityMapDashboard.java
├── images/
│ ├── delhi.png
│ ├── mumbai.png
│ ├── kolkata.png
│ ├── chd.png

---

## ▶️ How to Run
1. Open the project in any Java IDE (Eclipse / IntelliJ IDEA)  
2. Compile the project  
3. Run `CityMapDashboard.java`  
4. Click on cities to view crime data
